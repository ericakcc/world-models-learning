# **世界模型 (World Models) 研究與實作指南**

## **一、 建議閱讀論文清單 (按技術演進)**

以下論文根據技術類別與重要性排序，建議依序研讀以建立完整的知識體系。

| 類別 | 論文名稱 | 核心技術 | 對 RTX 5090 的研究意義 |
| :---- | :---- | :---- | :---- |
| **奠基理論** | [World Models](https://arxiv.org/abs/1803.10122) | VAE \+ RNN \+ MDN | 建立 VMC 基礎範式，適合理解「夢境訓練」的原始概念。 |
| **當前主流** | [DreamerV3](https://arxiv.org/abs/2301.04104) | Discrete Latent (RSSM) | 目前最穩定的 RL 框架，5090 可支持更大規模的 Buffer 與長序列訓練。 |
| **交互生成** | [Genie](https://arxiv.org/abs/2402.15391) | Video Transformer | 學習 Latent Action。5090 的算力足以支撐 Transformer 的高頻推理。 |
| **神經引擎** | [GameNGen](https://arxiv.org/abs/2408.14859) | Diffusion Model | 展示如何用擴散模型取代遊戲引擎，適合測試 5090 的 Tensor Core 加速。 |
| **物理特徵** | [V-JEPA](https://arxiv.org/abs/2404.08471) | Feature Prediction | Meta 提出的路徑，強調不預測像素而預測特徵，極度節省 VRAM。 |
| **具身駕駛** | [GAIA-1](https://arxiv.org/abs/2309.17080) | Multi-modal World Model | 自動駕駛世界模型的標竿，適合學習如何整合文字、動作與視頻。 |

## **二、 適合 RTX 5090 實作的資料集推薦**

RTX 5090 擁有 32GB VRAM，這讓您可以跑動一些傳統 30/40 系列顯卡較為吃力的資料集。

### **1\. 入門級：Gymnasium / Atari (基礎驗證)**

* **名稱**：CarRacing-v2, Atari 100k  
* **特點**：解析度低 (64x64 或 96x96)，數據量小。  
* **實作建議**：用來跑通 **DreamerV3** 的代碼，測試 VAE 壓縮與控制器 (Controller) 的訓練。在 5090 上，您可以在幾小時內完成訓練。

### **2\. 進階級：MineDojo (具身智慧)**

* **名稱**：[MineDojo](https://minedojo.org/) / Minecraft Data  
* **特點**：包含 730,000+ 小時的 Minecraft YouTube 影片及 Wiki 數據。  
* **實作建議**：實作類似 **STEVE-1** 的模型。5090 的大顯存可以容納較大的視訊編碼器，適合練習從未標記影片中學習動作指令。

### **3\. 專業級：nuScenes / OpenDV-2K (自動駕駛)**

* **名稱**：[nuScenes](https://www.nuscenes.org/) (Mini 或 Full 套裝)  
* **特點**：包含雷達、鏡頭、IMU 等多模態數據。  
* **實作建議**：實作簡化版的 **GAIA-1** 或 **UniAD**。32GB VRAM 足以支持多攝像頭 (Multi-view) 的輸入處理，這是練習「空間一致性」的最佳資料集。

### **4\. 頂級挑戰：YouTube-8M Subsets (生成式預訓練)**

* **名稱**：YouTube-8M / OpenDV-YouTube  
* **特點**：原始、雜亂的真實世界影片。  
* **實作建議**：實作 **Genie** 的時空 Transformer (Spatio-Temporal Transformer)。您可以挑選特定類別（如「駕駛」或「跑酷」）進行預訓練，測試模型對物理規律的自監督學習能力。

## **三、 硬體實作建議 (RTX 5090 專屬)**

1. **資料類型優化**：務必使用 bfloat16 或 float8\_e4m3fn 進行訓練，RTX 5090 對這些格式有硬體級優化，能翻倍提升訓練速度並減少顯存佔用。  
2. **併發環境**：利用 5090 的多核特性，在強化學習中開啟多個平行環境 (Parallel Envs)，可以大幅縮短收集經驗的時間。  
3. **顯存快取**：建議將資料集放在 Gen5 NVMe SSD 上，因為 5090 處理速度極快，磁碟 IO 往往會成為唯一的瓶頸。
